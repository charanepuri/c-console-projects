# C Console Projects in C

A collection of console-based applications built using the C programming language to practice and demonstrate fundamental programming concepts.

## 📚 Projects Included

### 1. Student Management System
A console application for managing student records with file handling support.

#### Features
- Add Student Records
- View All Students
- Search Student by ID
- Update Student Information
- Delete Student Records
- Save and Load Data from Files

#### Concepts Used
- Structures
- Arrays
- Functions
- File Handling
- Loops and Conditional Statements

**Repository:**  
https://github.com/charanepuri/StudentManagementSystem

---

### 2. Advanced Calculator
A menu-driven calculator application that performs various arithmetic operations.

#### Features
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Power Calculation
- Square Root
- Percentage Calculation

#### Concepts Used
- Functions
- Switch Statements
- Header Files
- Modular Programming
- Mathematical Operations

**Repository:**  
https://github.com/charanepuri/advanced-calculator-c

---

### 3. Number Guessing Game
An interactive game where players guess a randomly generated number and track their performance.

#### Features
- Random Number Generation
- Multiple Difficulty Levels
- Score System
- Leaderboard
- Achievement System
- Replay Option

#### Concepts Used
- Random Number Generation
- Functions
- File Handling
- Modular Programming
- Arrays and Strings

**Repository:**  
https://github.com/charanepuri/number-guessing-game-c
